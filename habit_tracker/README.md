Habit Tracker (SwiftUI)


Habit Tracker is a simple iOS application for tracking daily habits and monitoring progress.

Built to practice working with SwiftUI, MVVM architecture, and local data persistence.

```markdown
Features:
    Create new habits
    Mark a habit as completed for today
    Track total completion count
    Delete habits with swipe gesture
    Automatic local data persistence
```

```markdown
Tech Stack:
    Swift 5
    SwiftUI
    MVVM Architecture
    UserDefaults
    JSON Encoding/Decoding
    iOS 16+
```


```markdown
How It Works:
    User creates a new habit via text input
    Habits are stored locally using JSON encoding
    Completion status is tracked by storing dates
    Tapping the circle marks/unmarks today’s completion
    All changes are automatically saved to UserDefaults
```
