# MarketPulse

Приложение для отслеживания рыночных идей: список активов, обзор дня, сектора, детали актива и избранное

## Реализовано:

- экран обзора дня
- список активов с фильтрацией по типу
- поиск по тикеру и названию
- экран деталей актива
- экран секторов рынка
- избранное

## Экраны

1. `Обзор`
   лидеры дня, быстрые действия и выделенная идея недели
2. `Активы`
   акции, ETF и крипта с поиском и фильтрами
3. `Детали актива`
   цена, изменения, тезисы и риски
4. `Сектора`
   краткая сводка по направлениям рынка
5. `Избранное`
   сохраненные активы для дальнейшего отслеживания

## Стек

- `UIKit`
- `UINavigationController`
- `UITabBarController`
- `UICollectionView`
- `UITableView`
- `UserDefaults`


1. графики цены актива за день / неделю / месяц
