# Habit Tracker

Учебный проект на `FastAPI` для отслеживания привычек

В приложении можно добавлять привычки, отмечать выполнение за день, редактировать и удалять их, а также смотреть простую статистику. Есть веб-интерфейс и API

## Стек

- `Python`
- `FastAPI`
- `Jinja2`
- `HTML/CSS`

## Запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Запустить проект:

```bash
python3 -m habit_tracker.main
```

3. Открыть в браузере:

```text
http://127.0.0.1:8000
```

Swagger:

```text
http://127.0.0.1:8000/docs
```

## Структура

- `habit_tracker/main.py` - запуск приложения
- `habit_tracker/views/` - веб-маршруты
- `habit_tracker/api/` - API
- `habit_tracker/core/` - логика, модели, исключения
- `habit_tracker/templates/` - HTML-шаблоны
- `habit_tracker/static/` - стили и JS

## Что есть в проекте

- главная страница со списком привычек;
- страница отдельной привычки;
- отметка выполнения за текущий день;
- редактирование и удаление привычек;
- страница статистики;
- CRUD API для привычек

## Особенность

Данные сейчас хранятся в памяти, без базы данных. После перезапуска приложения всё сбрасывается

## Зачем делал

Этот проект делал как практику по `FastAPI`: хотел лучше понять роуты, шаблоны, обработку ошибок и структуру небольшого веб-приложения
