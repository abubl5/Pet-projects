# BrewBuddy

Приложение-компаньон для кофе: рецепты заваривания, таймер, дегустационные заметки и история последних сессий приготовления


## Реализовано

- список способов заваривания
- экран с деталями рецепта
- экран таймера приготовления
- экран отзыва после приготовления: вкус, оценка и заметки
- история последних сессий
- избранные рецепты

## Экраны

1. `Home`
   ежедневные рекомендации рецептов и быстрый доступ к категориям
2. `Methods List`
   таблица или коллекция способов заваривания
3. `Recipe Details`
   таблица или коллекция способов заваривания
4. `Brew Timer`
   таймер с пошаговыми этапами
5. `Session Review`
   форма с оценкой и вкусовыми заметками
6. `History`
   список предыдущих сессий приготовления
